# GSR Guest Portal Website  

This is the official Genesis Science Report (GSR) Guest Portal, where interview guests can submit materials and find resources.  

## Features  
- Submit materials form for guests  
- Responsive design  
- Clean, professional layout  

## Deployment  
The site is hosted using GitHub Pages. Visit the live site [here](https://gsrguestportal.com).  

## How to Run Locally  
1. Clone the repository  
2. Open `index.html` in your browser  

## Technologies Used  
- HTML  
- CSS  
- JavaScript
